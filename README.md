Enterprise FAQ agent that ingests business PDFs and answers domain specific queries using Hybrid Search and Cross Encoder Reranking. Built with FastAPI ChromaDB LangChain Pydantic Chainlit and Docker. Designed for real client deployment with persistent chat history and RAGAS evaluation.

## Enterprise RAG Platform
Production grade RAG system using Recursive Semantic Chunking and Cross Encoder Reranking for querying private PDF data. Achieved high faithfulness and answer relevancy scores via RAGAS evaluation framework. Tech: FastAPI ChromaDB LangChain RAGAS.

## Multi-Functional AI Agent
Autonomous agent with structured Tool Calling and JSON outputs using Pydantic v2 to handle complex multi step business workflows. Built with LangChain and optimized for multi agent cooperation.

## Rossmann Sales Predictor
Predictive model for Rossmann retail chain sales forecasting using feature engineering and ensemble methods. Achieved competitive accuracy on time series retail data. Tech: Python Scikit Learn Pandas.

## Churn Predictor
ML classification model to predict customer churn with data preprocessing feature selection and model evaluation using precision recall analysis. Tech: Python Scikit Learn Pandas FastAPI.

## Diabetes Risk Analyzer
Classification model deployed as an interactive Streamlit web app for diabetes risk prediction based on patient health parameters. Tech: Python Scikit Learn Streamlit.

## House Price Prediction
End to end regression pipeline for predicting house prices with data cleaning EDA and model comparison. Tech: Python Regression Scikit Learn Streamlit.

## Cat vs Dog CNN Classifier
Deep Learning image classification model using Convolutional Neural Networks with PyTorch deployed as an interactive Streamlit app. Tech: PyTorch CNN Streamlit.
